CRUD App Demo

This is a basic CRUD application built using Java and Spring Boot for the back-end and using Javascript, CSS and HTML for the front-end. This application allows the user to create, read, update, and delete data entries in a persistent H2 database. The Id field is automatically incremented as new entries are created. Updates and deletions work by specifying an Id to use as an endpoint. The application was deployed to a free cloud service and containerized using Docker.
